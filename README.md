# Quy Hoạch Sông Hồng - Dữ liệu & Bản đồ

Dự án cung cấp thông tin quy hoạch, giải phóng mặt bằng ven sông Hồng (Hà Nội).

## Thành phần hệ thống

- **Frontend**: GitHub Pages + Leaflet.js
- **API Layer**: Google Apps Script
- **Database**: Google Sheets
- **Automation**: OpenClaw Agent (AI-driven)

## Cấu trúc thư mục

- `index.html`: Giao diện chính và bản đồ.
- `configs/openclaw.json`: Cấu hình cho AI Agent thu thập dữ liệu.
- `SystemStructure.md`: Tài liệu kiến trúc chi tiết.

## Cài đặt & Vận hành

Chi tiết xem tại [SystemStructure.md](SystemStructure.md).
