# DULIEUQUYHOACH.COM - Hệ thống Tra cứu Quy hoạch

Dự án cung cấp nền tảng tra cứu quy hoạch thông minh, minh bạch, tập trung vào lợi ích của người dân.

## 🧬 DNA của dự án (5 Giá trị cốt lõi)

1.  **Người dân là trung tâm (User-Centric)**: Biến dữ liệu PDF phức tạp thành thông tin hành động "Nhà tôi có bị quy hoạch?".
2.  **Kiến trúc Siêu tinh gọn (Ultra-Lean)**: Vận hành 0 đồng trên GitHub Pages + Google Apps Script.
3.  **Hệ sinh thái 360 độ**: Dashboard tích hợp Bản tin, Tiến độ và Hỏi đáp pháp lý.
4.  **Tài chính Minh bạch**: Công cụ tính toán bồi thường dựa trên đơn giá pháp lý thực tế.
5.  **Thẩm mỹ Premium**: Ngôn ngữ thiết kế hiện đại, Inter font, Glassmorphism và Fly-to interactivity.

## 🏗️ Thành phần hệ thống

- **Frontend**: GitHub Pages + Leaflet.js
- **Automation**: Google Apps Script (GAS) tự động quét RSS và đồng bộ GitHub API.
- **Database**: JSON-based Engine lưu trữ trực tiếp trên GitHub Repo.
